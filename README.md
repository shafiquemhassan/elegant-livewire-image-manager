# Livewire Product CRUD

A simple and clean Product CRUD application built with **Laravel Livewire**.

This project demonstrates real-world Livewire concepts like:
- Add / Edit / Delete using a single modal
- Image upload with preview
- Search and pagination
- Bootstrap modal handling
- No page reloads

---

## Features

- Product CRUD
- Single modal for add & edit
- Image upload with preview
- Image update and delete handling
- Search products
- Pagination
- Flash messages
- Clean UI with Bootstrap

---

## Tech Stack

- Laravel 12
- Livewire
- Bootstrap 5
- MySQL

---

## Installation

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/livewire-product-crud.git

2. Go to project directory
cd livewire-product-crud

3. Install dependencies
composer install

4. Copy environment file
cp .env.example .env

5. Generate app key
php artisan key:generate

6. Configure database in .env
7. Run migrations and seed data
php artisan migrate --seed

8. Create storage link
php artisan storage:link

9. Start the server
php artisan serve
